# Voter Dashboard

## Overview

The voter dashboard provides registration, voting, and status tracking.

---

## 1. Dashboard Features

- Registration progress
- Biometric enrollment status
- Vote casting
- Vote confirmation
- Election information

---

## 2. Registration Flow

```
1. Enter National ID
2. Fill personal details
3. Upload face photo
4. Scan fingerprints
5. Set password
6. Submit registration
```
